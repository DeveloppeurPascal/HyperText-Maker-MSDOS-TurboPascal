# HyperText Maker - Turbo Pascal - MS-Dos - 1991

[This page in English.](README.md)

Programme de génération de fichiers HTD contenant des textes avec hyper liens.

Inspiré des textes d'aide fournis avec [Turbo Vision](https://fr.wikipedia.org/wiki/Turbo_Vision) pour prendre en charge les liens entre les pages de documentation, ce programme a été créé en 1991 pour encapsuler des fichiers textes vers des fichiers binaires et vice versa.

Bien entendu il a été développé sous [Turbo Pascal](https://fr.wikipedia.org/wiki/Turbo_Pascal).

Il est possible que ce projet soit compilable avec [Free Pascal](https://www.freepascal.org) mais je n'ai pas essayé. Les unités non fournies se trouvent dans le dépôt [DeveloppeurPascal/librairies-TurboPascal](https://github.com/DeveloppeurPascal/librairies-TurboPascal). S'il en manque faites le moi savoir pour que je les y ajoute.

Vous trouverez les exécutables (8 bits) dans le dossier "release". C'est la version du 17 avril 1995. Vous ne pourrez l'exécuter que sur un émulateur MS-Dos, Windows 3.x ou 95. Inutile d'essayer sur Windows au delà de ces versions.

Découvrez [mes autres projets open source en Turbo Pascal](https://github.com/DeveloppeurPascal?tab=repositories&q=TurboPascal&type=&language=&sort=) si l'archéologie logicielle vous branche.
